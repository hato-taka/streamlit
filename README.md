# streamlit
アプリの起動
`streamlit run app.py`

テキスト表示
```python
st.title("title") # タイトル
st.header("header") # ヘッダー
st.write("write") # 表示
st.markdown("# markdown") # マークダウンで表示
st.text("text") # テキスト表示
```
